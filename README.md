# CX/DX 통합 자가진단 애플리케이션

## PDF 다운로드 기능

진단 결과를 PDF 파일로 저장할 수 있습니다.

### 사용 방법

1. 진단을 완료하고 결과 화면으로 이동합니다
2. "결과 다운로드" 버튼을 클릭합니다
3. PDF 파일이 자동으로 브라우저의 기본 다운로드 폴더에 저장됩니다

### 저장 위치

PDF 파일은 브라우저의 기본 다운로드 폴더에 저장됩니다:
- Windows: `C:\Users\[사용자명]\Downloads\`
- 사용자 설정에 따라 다를 수 있습니다

### 파일명 형식

`CX-DX진단결과_YYYY-MM-DD.pdf`

예: `CX-DX진단결과_2025-12-12.pdf`

### 기술 사양

- **라이브러리**: jsPDF 2.5.1
- **용지 크기**: A4 (210mm × 297mm)
- **해상도**: 고해상도 (2x scale)
- **형식**: PDF (Portable Document Format)

### 주의사항

1. **브라우저 호환성**: 최신 버전의 Chrome, Edge, Firefox, Safari에서 정상 작동합니다
2. **팝업 차단**: 브라우저의 팝업 차단 설정을 확인해주세요
3. **다운로드 권한**: 브라우저가 파일 다운로드 권한을 요청할 수 있습니다

### 파일 구조

```
cxtest/
├── index.html          # 메인 HTML 파일
├── styles.css          # 스타일시트
├── data.js            # 질문 및 데이터
├── app.js             # 메인 애플리케이션 로직
└── pdf-download.js    # PDF 다운로드 기능 (NEW!)
```

### 문제 해결

**PDF가 다운로드되지 않는 경우:**
1. 브라우저 콘솔(F12)에서 오류 메시지 확인
2. 팝업 차단 해제
3. 다른 브라우저에서 시도
4. 페이지 새로고침 후 재시도

**PDF 품질이 낮은 경우:**
- pdf-download.js 파일의 `scale` 값을 조정하세요 (현재: 2)
- 값을 높이면 품질이 향상되지만 파일 크기가 커집니다
